# Sort Summary

## 目标

1. 说出10种以上的排序算法, 并说明它们的原理
2. 10分钟内实现任意一种排序算法

## 比较

|              |  **时间复杂度**  | **是否为稳定排序** | **是否是原地排序** |备注|
| :----------: | :--------------: | :----------------: | :----------------: | :---:|
| **冒泡排序** |    **O(n^2)**    |       **√**        |       **√**        |		|
| **选择排序** |    **O(n^2)**    |       **×**        |       **√**        |		|
| **插入排序** |    **O(n^2)**    |       **√**        |       **√**        |		|
| **希尔排序** | **O(n(logn)^2)** |       **×**        |       **√**        |		|
| **归并排序** |   **O(nlogn)**   |       **√**        |       **×**        |		|
| **快速排序** |   **O(nlogn)**   |       **×**        |       **√**        |		|
|  **桶排序**  |     **O(n)**     |       **√**        |       **×**        |		|
| **计数排序** |     **O(n)**     |       **√**        |       **×**        |counting sort 适合于重复数字多的情况	|
| **基数排序** |     **O(n)**     |       **√**        |       **×**        |radix sort 按个位, 十位, 百位一个个排序	|
|  **堆排序**  |   **O(nlogn)**   |       **×**        |                    |		|
